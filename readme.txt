Para rodar esse projeto é necessário ter o banco de dados MySQL 5(ou superior) instalado.
Antes da execução do projeto, é necessário rodar o script "labtrans.sql"(está no projeto) pra criar o banco de dados e suas tabelas.
O projeto foi testado com Tomcat 7(ou superior) e Java 7(ou superior) e 
pode ser executado pelo Eclipse ou realizar o deploy do arquivo "labtrans.war"(está no projeto) no Tomcat do servidor.
Para acessar o Sistema de Reserva de Salas deve-se usar o endereço http://"dominio":8080/labtrans
